# Cooking Notes — Personal Recipe Collection

**Cooking Notes** is a clean, ad-free, bilingual recipe website. It contains my personally tested recipes, original photos, and practical cooking notes — all in one place, always at hand.

Whether you're in the kitchen or at the grocery store, the site is fully adapted for all devices (mobile, tablet, desktop) and designed to help you cook with minimal distractions.

## ✨ Features

- 🍽️ **Personal recipes** — only dishes I’ve cooked and tested myself
- 📸 **Original photography** — real photos of finished meals
- 🌍 **Bilingual** — full support for Russian and English
- 📱 **Responsive design** — works perfectly on any screen size
- 🚫 **No ads** — just recipes, nothing else
- ⚡ **Fast & lightweight** — no unnecessary scripts or clutter
- 🛒 **Shopping-friendly** — easy to open on the go, check ingredients, and cook

👉 [glafiravankova.github.io/cooking-notes](https://glafiravankova.github.io/cooking-notes)

## 📂 Structure

- `index.html` — main page with category tabs (breakfast, soups, mains, desserts, etc.)
- `translations.js` — Russian/English language switcher
- `script.js` — tab switching and translation logic
- `style.css` — responsive styling for all devices
- `recipes/` — individual recipe pages
- `site-photos/` — original food photography

## 🛠️ Tech Stack

- HTML5 / CSS3 / JavaScript (vanilla)
- Font Awesome icons
- GitHub Pages for hosting

## 📱 Use Case

- Read recipes while cooking — no pop-ups, no distractions
- Check ingredient lists in the store — mobile-friendly layout
- Share recipes with friends — bilingual support
